# React Tourist Attraction Mini Project

A mini React project showcasing tourist attractions with an interactive and user-friendly interface.

## 🌟 Features

- Browse tourist attractions
- View detailed information about each attraction
- Responsive design for mobile and desktop
- Interactive user interface

## 📋 Prerequisites

- Node.js (v14 or higher)
- npm or yarn

## 🚀 Installation

1. Clone the repository:
```bash
git clone https://github.com/PrismBKK/react-tourist-attraction-mini-project-PrismBKK.git
cd react-tourist-attraction-mini-project-PrismBKK
```
2. Navigate to the project directory:
```bash
npm install
```
3. Install dependencies:
```bash
npm start
```
## 🎮 Usage
Start the development server:
```bash
npm start
```
The application will open in your browser at http://localhost:3000
